  • Advantage and Disadvantage

    • C#

    Used by employers in Information Technology, Engineering, Design, Professional Services, Management and Quality Control. Major Organizations are Microsoft Intel and Hewlett Packard. C# helps developers create XML web services and Microsoft. Used to develop a wide range of software applications that accompany Visual Studio IDE. The only language used by the Windows Presentation Foundation (WPF). It’s similar to Java in capabilities, Ideal for beginners and the go-to for working on Microsoft apps. C# was developed in 2000. C# was based off of C, C++, and Java languages.Example:

    Objective C

    Used by employers in Information Technology, Engineering, Management, Healthcare and Design. Increased flexibility with dynamic typing, often used along, side a framework such as Cocoa or Cocoa Touch and great first language for beginning programmers. Objective-C was developed at Apple in 1983 to address deficiencies in C language.Example:

    PHP (Hypertext Preprocessor)

    Used across industries, to include Information Technology, Engineering, Design, Professional Services, Healthcare, Management and Finance. Its Major Organizations are Facebook, Yahoo, CyberCoders and NextGen. PHP is part of the LAMP platform used by Facebook and Yahoo. It’s Easy to learn (as simple as embedding code inside HTML) and Free and open source. It can be used on all major operating systems and web servers. PHP was released in 1995 as a server-side scripting language, which is processed on a server to become a website in plain HTML.Example:

    SQL (Structured Query Language)

    SQL is used by most companies and organizations that gather data. Examples include Robert Half Technology, Nigel Frank, CyberCoders and UnitedHealthCare. SQL is used most in Data Analysis and Big Data Mining. SQL was developed at the IBM Research Center in 1974 and originally known as SEQUEL. The first commercial version was introduced in 1979 by ORACLE.Example:
